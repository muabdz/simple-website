How to deploy React app via netlify:

1. prepare project files, and push it to a github repo.
2. sign up and login to netlify.com
3. in the team overview menu, click on add new site / add new site from git.
4. Select, Connect and authorize to one of the Git provider.
5. Pick a repository from the connected github account.
6. select branch to deploy, and fill build command field with "npm run build", and publish directory field with "build".
7. click deploy site button and wait until the app is deployed
